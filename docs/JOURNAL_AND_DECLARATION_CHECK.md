# Journal And Declaration Check

## Practical assessment
- `Ethics approval and consent to participate`: substantively sufficient for a BMC-style declaration because it identifies the committee, approval number, Helsinki compliance, and consent waiver.
- `Consent for publication`: acceptable as long as the manuscript contains no identifiable patient images or case-level details.
- `Availability of data and materials`: the manuscript now includes a detailed statement mentioning the code package as Additional file 2 and a public DOI-minting repository. This is substantively sufficient for BMC.

## Current package-aware recommendation
The data-availability statement should explicitly distinguish:
- non-public raw patient-level files
- shareable code and aggregate outputs
- the route for controlled access requests
